# 迷宫视觉生成器（MAZE → FORM）产品说明

## 1. 产品定位

**迷宫视觉生成器**是一款在浏览器中运行的轻量创意工具：用户先为四个方向键配置「视觉动作」，再在随机生成的迷宫中用方向键行走；每一步都会把对应动作施加到预览画布上的全部图形（以及可选的叠加文字），从而把**迷宫路径**转化为**独一无二的抽象画面**。

核心价值：

- **规则先于结果**：同一套规则下，不同走法会生成不同视觉轨迹。
- **零安装**：打开 `index.html` 即可使用（需现代浏览器）。
- **可导出、可存档**：支持 PNG 静图、WebM 录像，以及到达终点时的画面自动收入图库。

---

## 2. 界面结构

页面大致分为三个区域：

| 区域 | 说明 |
|------|------|
| **左侧 · 迷宫面板** | 迷宫画布（逻辑尺寸 480×480）、开始/重开/图库入口、状态提示与操作说明。 |
| **中间 · 规则与资源** | 图形库、颜色、文字、方向键映射、材质与混合、背景、导出与回放控制。 |
| **右侧 · 预览面板** | 实时视觉画布（逻辑尺寸 640×480），与迷宫行走同步更新。 |

页眉可切换 **中文 / English**，界面文案会随之切换；部分设置会持久化到浏览器本地。

---

## 3. 推荐使用流程

1. **（可选）** 在中间栏调节图形库、颜色、材质、背景、方向键映射等；未开局时即可调整，参数会尽量保留。
2. 点击 **「开始新游戏」**：生成新迷宫，重置本局记录与图形初始状态（具体是否保留你的滑块/选项以页面逻辑为准；迷宫与玩家位置会更新）。
3. 使用键盘 **↑ ↓ ← →** 在迷宫中移动；每步对应一个已映射的「动作」，作用于**所有**当前图形。
4. 抵达终点后本局结束，可 **回放** 本局路径、**下载 PNG** 或 **下载 WebM**；终点画面可进入 **图库** 长期保存与管理。

**「重新随机迷宫」**：在保持你当前规则与外观设置的前提下，重新生成迷宫拓扑（具体是否与「新游戏」完全等价以实际逻辑为准，通常用于只想换迷宫结构时）。

## 4. 功能说明

### 4.1 迷宫与操作

- **迷宫**：基于网格的墙结构，起点与终点固定在对角区域习惯用法（以程序为准），玩家用方向键一格一格移动。
- **状态栏**：提示是否已开始、当前步数、是否完成、是否正在回放、视频导出进度等。

### 4.2 图形库与颜色

- **图形库（多选）**：可从多种基础图形中勾选一种或多种（如圆、方、三角、菱形、星形、六边形、线、螺旋等）。每种被选中的图形会在预览区生成**独立实例**，同步接受方向键动作。
- **每种图形的颜色**：在调色板中为当前库内图形分别指定颜色。
- **「随机颜色」**：为当前已勾选图形库中的每一类图形随机生成一组饱和度较高的颜色，并刷新显示；会参与本地设置的自动保存。

### 4.3 图形位置

- **居中在画面**：图形围绕预览区中心呈环状分散排布。
- **随机分布**：在画布安全范围内随机放置各图形实例；可多次点击以重新随机位置。

### 4.4 迷宫字形（可选）

- 在文本框中输入简短文字（有长度上限），文字渲染在预览区**安全边距内**，并随与方向键相同的动作体系发生缩放、旋转、色相、透明度等变化（与图形规则一致、但作用于文字层）。
- **文字位置**：居中或随机偏移（在安全区内）。
- **「更多」折叠区**：字号比例、文字描边开关与颜色、描边粗细；以及图形描边相关选项（与中间栏主流程中的描边设置对应，便于集中调节）。

### 4.5 方向键行为映射

为 **上 / 下 / 左 / 右** 各自指定一个动作。可选动作包括但不限于：

| 类型 | 含义（简述） |
|------|----------------|
| 旋转 ±15° | 改变图形朝向 |
| 水平/垂直拉伸、整体压缩/放大 | 改变比例与体量 |
| 色相偏移、透明度增减 | 色彩与通透度 |
| 随机抖动、位移 | 位置扰动 |
| 增强拖尾、脉冲 | 动态与轨迹质感 |
| **分裂（随机位置复制）** | 从现有图形复制出**第二个实例**，落在预览画布**随机合法位置**（有数量上限，防止过多实例拖慢性能） |

> 说明：每按一次方向键，该键所映射的动作会施加到**画面上每一个**图形实例。

### 4.6 材质、叠加与尺寸

- **材质**：如纯色、渐变、胶片颗粒、霓虹、线框、涂抹等，改变绘制风格。
- **叠加方式**：如正常、滤色、正片叠底、叠加、差值、线性减淡等，控制多图形与轨迹层之间的混合。
- **初始图形大小**：新开局时图形尺寸的基准。
- **胶片颗粒强度**：在适用材质下调节颗粒感强弱。

### 4.7 图形描边

可为图形开启描边，并设置描边颜色与线宽（具体控件以页面「更多」或主面板为准）。

### 4.8 背景设置

- **背景色**、**透明背景**。
- **背景图片 / 背景视频**（本地文件选择）；视频通常静音循环播放作为底图氛围。

### 4.9 记录回放

- 完成一局后，可使用 **首帧 / 上一帧 / 播放·暂停 / 下一帧 / 末帧** 控制时间轴。
- **回放速度**：滑块调节播放快慢。
- **回放静音**：降低回放时与方向相关的提示音对环境的干扰（若实现含音效）。

### 4.10 导出

- **下载图片（PNG）**：导出当前预览画面为位图。
- **下载视频（WebM）**：按本局记录回放并编码为视频文件（导出过程中状态栏会提示进度；耗时与机器性能、步数有关）。

### 4.11 终点图库

到达终点并完成一局后，**最终一帧画面**可自动保存到浏览器 **IndexedDB** 图库中。

图库内支持：

- **一键清空**：删除全部已存终点图（需确认）。
- **勾选缩略图**后：**导出所选**、**删除所选**（删除需确认）。
- **一键导出**：导出图库中全部图片；优先尝试打包为 **ZIP**（依赖在线加载的打包库）；若失败则退化为**逐个文件下载**，并可能给出提示。

点击缩略图可在新标签页查看较大图片。

---

## 5. 数据与隐私

- **本地设置**：规则、颜色、部分 UI 选项等会写入浏览器 **localStorage**，便于下次打开恢复。
- **图库**：仅存于本机浏览器的 **IndexedDB**，不上传服务器；清除站点数据会导致图库丢失。
- **背景图/视频**：仅在本地内存与画布中使用，不会自动上传。

---

## 6. 环境与限制（须知）

- 请使用较新的 **Chrome / Edge / Firefox / Safari** 等支持 HTML5 Canvas、IndexedDB、`MediaRecorder`（WebM 导出）的浏览器。
- **WebM 导出**依赖浏览器编码能力，部分浏览器或操作系统组合可能失败或质量不同。
- **图库 ZIP 导出**依赖从 CDN 加载 JSZip；离线或网络受限时可能退化为逐个下载。
- 图形实例数量有上限（防止性能问题）；**分裂**动作在接近上限时不再产生新实例。
